Welcome to the Optimity Advisors Skill Bank!

What is a skill bank?

A skill bank is a single source to inventory knowledge, skill, expertise and experience so we as a company may easily and effectively place resources where needed.

Is the site functional?

This is only a prototype built using Angular to help demonstrate the idea of an online skill bank.

Where is the data coming from?

As of right now, the data is being driven from a fake API and the values do not reflect true accurate data. Once again, the data is for testing purposes only and does not reflect actual statistics.
